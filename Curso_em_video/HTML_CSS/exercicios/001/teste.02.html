Guia de como adcionar simbolos e figuras ao seu site usando HTML5.

muitas vezes quando iniciamos a criação de um site queremos adicionar alguns simbolos como por exemplo: o simbolo matemático Δ ou o $

mas a pergunta é:

como colocar?

Acredite é mais facíl do que parece.
1:tenha um estudo previo sobre o HMTL5. 2:tenha um softwer para escrever seu código.3:abra uma pasta para testar e após isso coloque o início obrigatorio para começar seu código HTML5. 4:após as modificações opcionais para colocar um simbolo é bem simples. 5:basta escrever (&) e o nome do simbolo como (delta) ou (dollar). 6:Onde ficaria &dollar; 7:E para fechar uma dica final, para simbolos sempre adicione (;) ao final.
